1. 概要

本アプリは、色付きの牌が並んだグリッド盤面において、
行または列を入れ替え・回転させ、色を揃えて消すパズルゲームである。

落ちゲー要素・連鎖・特殊アイテム・演出（色・音）を重視し、
触って気持ちいい操作感と連鎖の爽快感を目的とする。

対応プラットフォーム：
	•	iOS
	•	Android
	•	Web（可能なら）

開発技術：
	•	Flutter
	•	Flame（ゲームループ・描画用）

⸻

2. ゲーム基本仕様

2.1 盤面
	•	盤面サイズ：7 × 7
	•	各マスには色付きの牌（タイル）が存在
	•	色の種類：5〜6色
	•	上から新しい牌が落下して補充される

例：

A B C D E F G
1 🟩🟦🟨🟧🟧🟪🟨
2 🟨🟩🟦🟪🟪🟨🟦
3 🔵🔵🟣🟠🟢🟡🟠
...


⸻

3. 操作仕様

3.1 行・列の入れ替え

プレイヤーは以下の操作が可能：
	•	行を選択して別の行と入れ替え
	•	列を選択して別の列と入れ替え

例：
	•	2列目と3列目を入れ替え
	•	4行目と5行目を入れ替え

⸻

3.2 行・列の回転

選択した行または列は 90度回転 可能。

制限：
	•	回転は1ターンに1回まで
	•	または回数制限制（例：3回まで）

回転すると牌の並び順が変わる。

⸻

4. 消去ルール

4.1 基本消去

以下の条件で牌が消える：
	•	同じ色が 縦または横に3つ以上 並ぶ

消去後：
	•	上から牌が落下
	•	空いた場所に新しい牌が生成
	•	連鎖判定を行う

⸻

4.2 十字消し

以下の条件で特殊消去：
	•	同じ色で 十字型 が完成

例：

  ●
● ● ●
  ●

効果：
	•	十字部分が消える
	•	中央の牌が 特殊アイテム に変化

⸻

5. 特殊アイテム

例：

アイテム	効果
爆弾	周囲3×3を消す
縦消し	列をすべて消す
横消し	行をすべて消す
色消し	同色をすべて消す

特殊アイテムは十字消しや連鎖で生成される。

⸻

6. ゲーム進行

6.1 ターンの流れ
	1.	プレイヤーが行/列操作
	2.	消去判定
	3.	消去
	4.	落下
	5.	連鎖判定
	6.	新規牌生成
	7.	操作可能状態へ戻る

※ 消去を待たずに次操作できる仕様も検討

⸻

7. スコア
